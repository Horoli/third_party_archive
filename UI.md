# UI 작업 내용 (UI.md)

이 문서는 프로젝트의 사용자 인터페이스(UI)와 관련된 최신 변경 사항을 기록합니다.

## 1. 전역 동기화 및 언어 제어 (FAB 영역)
- **통합 정보 인터페이스**: 화면 우측 하단에 `[동기화 시간 위젯] [언어 전환 버튼]`을 나란히 배치했습니다.
- **WidgetNinjaSyncInfo**: poe.ninja와의 최신 데이터 동기화 시간을 전역적으로 표시합니다.
    - **디자인**: 반투명 검은색 배경의 깔끔한 칩 형태.
    - **상호작용**: 호버 시 "데이터는 poe.ninja와 연동됩니다" 툴팁 노출.
- **Floating Language Button**: 현재 언어(`KO`/`EN`)를 즉시 전환하며, 앱 내의 모든 반응형 UI가 즉각 대응합니다.

## 2. 갑충석 시세표 (Scarab Price Table)
- **직관적인 시세 버튼**: 기존의 긴 텍스트를 `40c+ [5]` 형식으로 단순화하여 공간 효율성과 가독성을 극대화했습니다.
- **다국어 툴팁 및 분류**: 이름뿐만 아니라 분류 정보(Class)도 영문/국문으로 완벽하게 전환됩니다.
- **정규식 복사 최적화**: 영문 모드 사용 시 검색창에 바로 사용 가능한 `"scarab "` 접두어가 포함된 정규식을 제공합니다.

## 3. 잔돈 계산기 (Change Calculator)
- **완전한 i18n 적용**: 모든 입력 필드 레이블, 버튼, 결과 텍스트가 다국어화되었습니다.
- **검증 UI**: 오류 메시지(값 입력 필요, 소수점 제한 등)도 선택된 언어에 맞춰 실시간으로 나타납니다.

## 4. 메인 홈 및 네비게이션
- **메뉴 간소화**: `PageRandomBuildSelector`를 메뉴에서 숨겨 현재 활발히 사용되는 기능 위주로 구성했습니다.
- **반응형 대응**: 세로/가로 모드에 따라 Drawer와 상단 바 구성을 최적화했습니다.

## 5. 날짜 표시 표준화
- 앱 내의 모든 데이터 업데이트 시간은 `yyyy-MM-dd HH:mm` 형식으로 표시됩니다.
